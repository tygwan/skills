## [2.1.0] - 2025-11-12

### Added
- New skill: **artifacts-builder** - Suite of tools for creating elaborate, multi-component claude.ai HTML artifacts using React, Tailwind CSS, shadcn/ui
- New skill: **brand-guidelines** - Applies Anthropic's official brand colors and typography to any artifact
- New skill: **theme-factory** - Toolkit for styling artifacts with 10 pre-set themes with colors/fonts
- New skill: **using-git-worktrees** - Creates isolated git worktrees with smart directory selection and safety verification
- New skill: **webapp-testing** - Toolkit for testing local web applications using Playwright
- New skill: **finishing-a-development-branch** - Guides completion of development work with structured options for merge, PR, or cleanup

### Changed
- Updated README.md with documentation for 6 new skills
- Updated marketplace.json to version 2.1.0
- Added plugin.json files for all new skills
- Organized skills alphabetically in marketplace.json
